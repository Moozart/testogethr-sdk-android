# Contributing

Thanks for contributing.

## What changes are welcome here

- Android SDK integration documentation improvements
- Release note/tag accuracy updates
- Reproducible bug reports and integration examples

## What is out of scope

- Requests for private SDK source code
- Source-level implementation changes (handled in private repositories)

## Before opening a PR

1. Confirm the issue is reproducible on the latest SDK release.
2. Keep docs examples aligned with current public API.
3. Link related issue(s) in the PR description.
